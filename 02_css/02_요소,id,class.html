요소 선택자

HTML 태그 이름을 선택자 이름으로 사용한다.

선택자는 우선 순위가 있다.

id, class, 요소 순으로 지정된다.

클래스 선택자

클래스와 아이디 이름은 지정 규칙은 아래와 같다.

첫 글자는 영문자, _, -로 지정한다.

두 번째 문자부터 영문자, _, -, 숫자로 지정한다.

영문의 대소문자를 구별한다.

클래스명은 같은 이름을 중복 사용할 수 있다.

HTML 클래스 속성의 값은 띄어쓰기로 여러 개 지정할 수 있다.

아이디 선택자

아이디명은 한 페이지 내에서 중복 사용할 수 없다.
